# 数据采集指南

## 数据采集工具

聆心（Lingxin）手语识别系统提供了专门的数据采集工具 `tools/collect_data.py`，用于采集手语数据。

## 采集步骤

1. **准备工作**
   - 确保摄像头正常工作
   - 安装所需依赖：`pip install -r requirements.txt`
   - 确保光线充足，背景简洁

2. **运行采集工具**
   ```bash
   python tools/collect_data.py
   ```

3. **输入采集信息**
   - 输入要采集的词语
   - 输入采集者ID
   - 输入采集样本数

4. **采集过程**
   - 摄像头会打开，显示实时画面
   - 按照提示做出相应的手语动作
   - 按 `s` 键保存当前帧的关键点数据
   - 按 `q` 键退出采集

5. **数据保存**
   - 采集的数据会保存在 `data/raw/collected/word_词语/` 目录下
   - 文件名格式：`{采集者ID}_{样本序号}.npy`

## 采集规范

1. **采集环境**
   - 光线充足，避免强烈背光
   - 背景简洁，避免干扰
   - 手部区域清晰可见

2. **采集动作**
   - 每个词语采集多个不同角度的样本
   - 动作要标准、连贯
   - 保持手部在画面中央

3. **采集数量**
   - 每个词语至少采集10个样本
   - 建议采集多个不同人的样本，以提高模型泛化能力

## 数据格式

采集的数据为 `.npy` 格式，包含以下信息：

- 手部关键点坐标
- 提取的特征向量

## 数据管理

- 定期备份采集的数据
- 对数据进行分类整理
- 标注数据质量，移除异常样本
