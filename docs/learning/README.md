# 聆心（Lingxin）手语识别系统 - 学习路径总览

> [!abstract] 项目简介
> 聆心是一个基于深度学习的手语识别系统，能够识别中国手语（CSL）中的孤立词汇。系统支持多种模型架构，包括传统机器学习模型（SVM、随机森林、MLP）和深度学习模型（LSTM、Transformer）。
