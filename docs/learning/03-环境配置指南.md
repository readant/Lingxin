# 环境配置指南

> [!info] 学习目标
> 1. 掌握 Python 环境管理方法
> 2. 能够成功安装项目依赖
> 3. 了解 GPU 加速配置（可选）

## 一、环境要求

### 1.1 基础环境

| 要求 | 说明 |
|------|------|
| 操作系统 | Windows 10/11, macOS, Linux |
| Python | 3.8 或更高版本 |
| 内存 | 建议 8GB 以上 |
| 硬盘 | 至少 10GB 可用空间 |

### 1.2 GPU 支持（可选）

如果需要进行深度学习训练，建议配置 GPU：

| 组件 | 最低要求 | 推荐配置 |
|------|----------|----------|
| NVIDIA GPU | GTX 1050 | RTX 3060 及以上 |
| 显存 | 4GB | 8GB 及以上 |
| CUDA | 11.3 | 11.8 |
| cuDNN | 8.x | 8.x |

## 二、环境搭建步骤

### 2.1 方案一：使用 conda（推荐）

```bash
# 1. 创建虚拟环境
conda create -n lingxin python=3.9

# 2. 激活环境
conda activate lingxin

# 3. 安装 PyTorch（GPU 版本）
conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia

# 4. 安装其他依赖
pip install -r requirements.txt
```

### 2.2 方案二：使用 pip venv

```bash
# 1. 创建虚拟环境
python -m venv lingxin_env

# 2. 激活环境
# Windows
lingxin_env\Scripts\activate
# Linux/macOS
source lingxin_env/bin/activate

# 3. 安装依赖
pip install -r requirements.txt
```

## 四、验证安装

try:
    import numpy
    print(f"✓ NumPy: {numpy.__version__}")
except ImportError as e:
    print(f"✗ NumPy: {e}")

try:
    import cv2
    print(f"✓ OpenCV: {cv2.__version__}")
except ImportError as e:
    print(f"✗ OpenCV: {e}")

try:
    import mediapipe
    print(f"✓ MediaPipe: 已安装")
except ImportError as e:
    print(f"✗ MediaPipe: {e}")

try:
    import torch
    print(f"✓ PyTorch: {torch.__version__}")
    print(f"  CUDA 可用: {torch.cuda.is_available()}")

    print(f"✗ scikit-learn: {e}")

## 五、GPU 配置（可选）

### 5.1 检查 NVIDIA 驱动

```bash
nvidia-smi
```

### 5.2 安装 CUDA 和 cuDNN

> [!warning] 重要提示
> CUDA 版本必须与 PyTorch 版本兼容！请访问 [PyTorch 官网](https://pytorch.org/) 查看支持的 CUDA 版本。

```bash
# 安装 PyTorch（指定 CUDA 版本）
pip install torch==2.0.0+cu118 torchvision==0.15.0+cu118 --extra-index-url https://download.pytorch.org/whl/cu118
```

### 5.3 验证 GPU 可用

```python
import torch

print(f"PyTorch 版本: {torch.__version__}")
print(f"CUDA 可用: {torch.cuda.is_available()}")

if torch.cuda.is_available():
    print(f"CUDA 版本: {torch.version.cuda}")
    print(f"GPU 数量: {torch.cuda.device_count()}")
    print(f"当前 GPU: {torch.cuda.get_device_name(0)}")
    print(f"GPU 内存: {torch.cuda.get_device_properties(0).total_memory / 1024**3:.2f} GB")
```

## 六、常见问题

### 6.1 MediaPipe 安装失败

**问题**：安装 MediaPipe 时报错

**解决方案**：

```bash
# Windows 用户可能需要安装 Visual Studio Build Tools
# 或者尝试使用预编译版本
pip install mediapipe --prefer-binary
```

### 6.2 PyTorch GPU 版本安装失败

**问题**：无法安装 GPU 版本的 PyTorch

**解决方案**：

```bash
# 1. 先卸载 CPU 版本
pip uninstall torch torchvision

# 2. 从 PyTorch 官方源安装
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```

### 6.3 OpenCV 导入错误

**问题**：`ImportError: cv2.cv2` 找不到

**解决方案**：

```bash
pip uninstall opencv-python
pip install opencv-python==4.8.0.74
```
